# Gigahertz 📡

**Gigahertz** is a Minecraft Forge mod that brings realistic cellular network evolution into the blocky world.  
Build your own mobile infrastructure, progress through five generations (1G to 5G), and experience how wireless communication transformed from simple analog voice to ultra-fast digital data.

---

## 📱 Current Status – Alpha 0.1.0

The mod is in **early Alpha**. The first generation (1G) is fully playable and stable.

### ✅ Implemented Features (1G – Analog Era)

- **1G Walkie‑Talkie**  
  Portable handheld radio for short‑range communication.  
  - **Direct Mode:** 32‑block radius – works anywhere, like a classic walkie‑talkie.  
  - GUI with a text input field and a `Send` button.

- **1G Antenna Tower**  
  The first base station of your network.  
  - Automatically relays messages from nearby walkie‑talkies, boosting their range to **64 blocks**.  
  - When a message is relayed, the tower plays a radio static sound and spawns electric spark particles.

- **Smart Signal Indicator**  
  The walkie‑talkie GUI shows a 4‑bar signal strength meter that changes colour (red → orange → yellow → green) depending on your distance from the nearest tower.  
  - Without a tower, the signal stays at 1 red bar – you're on the edge of connectivity!

- **Crafting Recipes**  
  Both the walkie‑talkie and the tower are craftable using vanilla materials (iron, redstone, copper, etc.).

---

## 🎮 How to Install

1. Install **Minecraft Forge 1.20.1** (version `47.4.10` or later).
2. Download the latest `.jar` file from the [Releases](https://github.com/YourErrorDream/Gigahertz/releases) page.
3. Place the `.jar` into your `.minecraft/mods` folder.
4. Launch Minecraft with the Forge profile and enjoy!

> ⚠️ **Note:** This mod is still in Alpha. Always backup your worlds before using it.
